# v4 Core Record Examples

This directory contains small reference examples for the v4 core record model.

Purpose:
- show the minimum viable shape of key v4 records
- provide stable examples for doc readers
- give future tests and migration work a lightweight fixture set

Included examples:
- `Inbox/`
- `Leads/`
- `Notes/`
- `Activities/`

These examples are illustrative only. They are not a real vault.
