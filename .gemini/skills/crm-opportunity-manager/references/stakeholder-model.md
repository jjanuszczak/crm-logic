# Stakeholder Model

Minimum expected structure:

- `primary-contact`: the main operator-facing person for the opportunity
- `influencers`: other known stakeholders with meaningful influence on the path

Body expectations:
- `Economic Buyer` should usually reflect the primary contact unless a different buyer is known
- `Influencers` should mirror the structured influencer set when known
- the rest of the stakeholder section can remain prose-first until the schema expands

Operational guidance:
- do not leave active opportunities without a clear primary contact
- add influencers when they materially affect qualification, proposal, or close path
- if no stakeholder coverage exists, `review` should call that out
